Dependency Information

Apache Maven

<dependency>
  <groupId>org.olympe</groupId>
  <artifactId>musicplayer</artifactId>
  <version>1.0</version>
</dependency>

Apache Buildr

'org.olympe:musicplayer:jar:1.0'

Apache Ivy

<dependency org="org.olympe" name="musicplayer" rev="1.0">
  <artifact name="musicplayer" type="jar" />
</dependency>

Groovy Grape

)

Grails
