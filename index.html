Get it now

Download the binaries of the newest version of Olympe Music Player and have a lot of fun using it, or get the source code and run it yourself.

For more informations about the current release, visit the release notes.

Get involved

Olympe Music Player is a free software and open source project. We will be happy to welcome new volunteers to work with. Just visit the contribute page to get more informations.

Documentation

This project website covers both the user documentation through tutorials, installation guide and a user guide; and a developer documentation through the maven project documentation and the project reports documentation.
